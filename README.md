# Rooms to Chat

A web app where you can chat with people online and make friends.
https://go-chat.duckdns.org/



## Features

- Real-time messaging
- Rooms
- Private 1-to-1 chats with friends

## Tech Stack

- Go
- PostgreSQL
- WebSocket
- HTML
- Hosted on AWS EC2
- UUID for user authentication
